# 🚀 Elton Santos | Backend Developer Portfolio

Este projeto é um portfólio dinâmico que integra tecnologias de ponta para demonstrar minhas habilidades em desenvolvimento Backend, automação e Inteligência Artificial.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, Tailwind CSS, JavaScript (ES6+).
- **Backend:** Node.js, Express.
- **IA:** Integração com Google Gemini Pro (IA Generativa).
- **Automação:** API do GitHub para carregamento dinâmico de projetos.

## 🛡️ Blindagem & Segurança
Para garantir a integridade e segurança do sistema, o projeto foi construído seguindo boas práticas de "blindagem" digital:
1. **Proteção de Chaves:** Uso de variáveis de ambiente (`.env`) para ocultar chaves de API.
2. **CORS Security:** Restrição de acesso ao backend apenas por origens autorizadas.
3. **Sanitização:** Tratamento de entradas de usuário para prevenir injeções básicas no chat.
4. **Rate Limiting:** Estrutura preparada para evitar abusos de requisições na IA.

## 🚀 Como Executar
1. Clone o repositório.
2. Na pasta `/backend`, execute `npm install`.
3. Crie um arquivo `.env` com sua `GEMINI_API_KEY`.
4. Inicie o servidor com `node server.js`.
5. Abra o `index.html` no seu navegador.
